Secure

In an age where collecting user metrics and personal information has become a common practice, your security and privacy are our number one concern. With dahliaOS, your data stays with you, away from prying eyes. Our sandboxed application container system makes sure that your data remains protected from malicious software and targeted attacks. Learn more about our security and privacy features.

Open Source

The source code to all of our code is free and available to the public, and licensed under the GNU General Public Licence v3, meaning you are allowed to make changes to it and redistribute it as much you want. If you run into a problem, you can report it to the developers on Github, Discord, or send us an email. We are always happy to provide help and documentation to our code. Learn more about DahliaOS and open source.

speed

Fast

By optimizing individual kernel builds for every device, dahliaOS provides a fast and power-efficient experience for every device, from smartphones to laptops, to high-end workstations. Through the use of two separate kernels, we can ensure support for every device. For other devices, we offer a monolithic build that supports every device. Learn more about system optimization
